FolderIconMaker

This script takes source images and converts them into a sleek looking image using a template DVD Case made by Manicho of deviantart. These images can then be used as Folder icons in Windows Vista or newer.

Written in Python

Author: @Zaexu, Zaexu

Requires: Python, Pillow, ImageMagick(Required to be installed on system)
